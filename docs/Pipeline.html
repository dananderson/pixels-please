toBuffer() → {Promise.<Buffer>}

Source:
  • output.js, line 23

Output image to a Buffer. All image processing occurs in a background thread that will not block Node's main loop. If the background thread pool is full, the operation will be queued until a thread is available.

The returned buffer has an extended field header of type Header, containing the image and Buffer details.

If no format is specified, the default format will be bytes.
